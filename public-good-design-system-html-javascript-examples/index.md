# Public Good Design System — HTML JavaScript Examples

Styled example pages demonstrating headless components with Plain HTML + vanilla JavaScript with NHS UK styling. Examples use NHS UK design system CSS for visual styling on top of headless components.

## Features

- Example pages demonstrating all major component families
- Plain HTML + vanilla JavaScript with NHS UK styling
- NHS UK design system CSS for consistent styling
- WCAG 2.2 AAA accessibility compliance
- Responsive design for mobile and desktop

## Architecture

- **Plain HTML + vanilla JavaScript with NHS UK styling** for page implementation
- **Headless components** imported from the corresponding headless library
- **NHS UK CSS** for visual styling
- **Accessible**: WCAG 2.2 AAA

## Testing

- Test framework: WebDriverIO
- Run tests: `npx wdio run wdio.conf.js`
